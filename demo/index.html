How To Use This Demo

Each card has a live preview first and a code editor underneath it. Change the HTML, wait a moment, and the date picker re-initializes with your new markup.

Quick Tips

  • Use ← → to move between segments in segmented mode.
  • Swipe left or right on touch devices while the calendar is open to change months.
  • For manual typing, set data-input-mode="native" and data-calendar="false".

Segmented Single-Date Picker

Hotel Booking Range Picker

Pick check-in and check-out in a single control with hotel-style range selection.

Turns off the popup calendar and lets the browser handle cursor movement, backspace, and typing naturally.

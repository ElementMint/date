@elementmints/date

A lightweight, dependency-free, attribute-driven date picker

<10 KB gzipped Zero dependencies HTML-first API Accessible

Basic Date Picker

Just add data-datepicker to any input. That's it.

<input type="text" data-datepicker placeholder="Select a date">

Custom Display Format

Use data-format to change how dates are displayed.

<input type="text" data-datepicker data-format="DD/MM/YYYY">

Min & Max Range

Restrict selectable dates with data-min and data-max.

<input type="text" data-datepicker data-min="2026-01-01" data-max="2026-12-31">

Dark Theme

Set data-theme="dark" for a dark calendar. Also supports "light" and "system".

<input type="text" data-datepicker data-theme="dark">

German Locale

Locale-aware month and day names via Intl.DateTimeFormat.

<input type="text" data-datepicker data-locale="de-DE" data-format="DD.MM.YYYY">

French Locale

Supports any BCP 47 locale tag.

<input type="text" data-datepicker data-locale="fr-FR" data-format="DD/MM/YYYY">

Required with Validation

Add data-required for form validation. Error messages render automatically.

<input type="text" data-datepicker data-required data-min="2026-01-01">

Weekdays Only

Custom rules via data-validate="weekday".

<input type="text" data-datepicker data-validate="weekday">

Future Dates Only

Combine rules: data-validate="future".

<input type="text" data-datepicker data-validate="future">

Disabled Specific Dates

Pass a comma-separated list via data-disabled-dates.

<input type="text" data-datepicker data-disabled-dates="2026-12-25,2026-12-26,2026-01-01">

Web Component

Use the <date-picker> custom element for encapsulated usage.

<date-picker data-format="DD/MM/YYYY" data-theme="system"> </date-picker>

JavaScript API

Programmatic control via the DatePicker class.

const picker = new DatePicker(document.getElementById('my-input')); picker.open(); picker.setValue('2026-06-15'); picker.getValue(); // "2026-06-15" picker.destroy();

## [0.0.0-development] - 2026-04-05

### Initial Development

- Project scaffolding and build system (Rollup + TypeScript + Sass)
- Core date picker with calendar popup and segmented input
- Auto-initialization via `data-datepicker` attribute
- Configuration through `data-*` attributes
- Date formatting and parsing (`DD`, `MM`, `YYYY` tokens)
- Min/max date range enforcement
- Disabled dates support
- Built-in validation rules (`weekday`, `future`, `past`)
- Async server-side validation via URL
- Keyboard navigation (arrow keys, Enter, Escape)
- Full ARIA support and screen-reader announcements
- Focus trapping within the calendar popup
- Light, dark, and system-preference themes
- Locale-aware month and day names via `Intl.DateTimeFormat`
- `<date-picker>` Web Component
- React adapter (`createReactDatePicker`)
- Vue 3 adapter (`createVueDatePicker`)
- Angular adapter (`createAngularDatePicker`)
- Plugin system
- IIFE bundle for CDN usage with auto-init
- ESM and CJS module exports
